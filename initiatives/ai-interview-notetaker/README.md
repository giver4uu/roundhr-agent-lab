# AI 면접 녹음 및 분석 (AI Interview Notetaker)

## Initiative 개요

면접관이 메모 작성 부담 없이 지원자와의 대화에 온전히 집중할 수 있도록 지원하고, 면접 대화 데이터를 온톨로지 지식 그래프로 구조화하여 채용 검증의 질을 근본적으로 향상시키는 AI 면접 노트테이커 기능

## 핵심 가치 제안

### 사용자 관점
1. **면접관 집중력 향상**: 필기 부담 제거로 지원자와의 대화에만 집중
2. **실시간 면접 지원**: 지원자 답변 기반 추가 역량 검증 질문 제안
3. **객관적 평가**: 주관적 기억이 아닌 실제 대화 기록 기반 분석 리포트
4. **시간 절감**: 피드백 작성 시간 64% 단축 (28분 → 10분)

### 라운드(전략적) 관점
1. **온톨로지 지식 그래프 구축**: 면접 대화(동적 데이터) 확보로 깊이있는 온톨로지 구조 구축
2. **채용 검증 고도화**: 정적 데이터(이력서, 평가) + 동적 데이터(면접 대화) 결합
3. **학습하는 채용 시스템**: 면접 패턴 학습으로 채용 성공 예측 및 질문 제안 품질 향상
4. **조직 채용 역량 축적**: 면접 데이터가 쌓일수록 더 정확한 인사이트 제공

- **면접관**: 필기 부담 없이 대화에 집중하고 싶은 면접관

## 성공 지표

### Tier 1: Product Engagement (즉시~1개월)
- AI 노트테이커 활성화율: 60%
- 면접당 사용률: 75%
- AI 요약 열람률: 90%

### Tier 2: Business Impact (1~3개월)
- 피드백 작성 시간: 28분 → 10분 (-64%)
- 피드백 제출률: 72% → 95% (+23%p)
- 디브리프 완료 시간: 42분 → 20분 (-52%)

### Tier 3: Long-term Success (3~6개월)
- 면접관 만족도: 3.8 → 4.5/5.0
- 지원자 면접 경험: 3.9 → 4.6/5.0
- 채용 품질 (90일 리텐션): 87% → 92%

### Tier 4: Business Metrics (6~12개월)
- 기능 리텐션: 85%
- Starter → Growth 전환: +12%p
- Enterprise Win Rate: 18% → 25%

## 핵심 가설

1. **H1**: 면접관이 필기 부담에서 해방되면 지원자와의 대화에 더 집중할 수 있고, 이는 면접 품질 향상으로 이어진다.
2. **H2**: 객관적 면접 기록은 주관적 기억보다 정확한 평가를 가능하게 한다.
3. **H3**: AI 기반 실시간 질문 제안은 면접관의 경험과 무관하게 일관된 고품질 면접을 가능하게 한다.
4. **H4**: 면접 대화 데이터(동적 데이터)를 구조화하면 채용 검증의 질이 향상되고, 이는 장기적으로 조직의 채용 역량이 된다.

## 기술 스택 (예상)

- **STT**: OpenAI Whisper (한국어/영어 전사)
- **LLM**: Claude 3.5 Sonnet (요약, 질문 제안, 분석)
- **온톨로지**: Neo4j 또는 커스텀 그래프 구조
- **화상회의 통합**: Zoom API, Google Meet API
- **저장**: AWS S3 (암호화 녹음), RDS (메타데이터)
